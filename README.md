# Atari Breakout
Projeto voltado para fins educacionais, com intenção de aprender javascript, focando nos conceitos das bibliotecas de javaScript.

## Aprendizados

Compreendi os conceitos das bibliotecas de javaScript.

## Funcionalidades

- use o mouse para movimentar uma raquete e com ela rebater a bola para quebrar os tijolos

## Stacks utilizada

**Front-end:** Html, JavaScript &biblioteca P5.

## Deploy

Para fazer o deploy desse projeto faça um clone do repositório e execute o arquivo em index.html.

```bash
  git clone https://github.com/MateusMengardaMcM/atari_breakout
```
